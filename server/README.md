# Server

## Technologies

- Fastify
- Eslint + Prettier
- TapJS (for testing)
- Prisma ORM
- PostgresSQL
- Typescript

## Available Commands

#### Run Development Server

```
yarn dev
```

#### Linter

```
yarn lint #just lint and point some errors
```

```
yarn lint:fix #lint and fix some errors
```

#### Styles

```
yarn style
```

```
yarn style:fi
```

#### Tests

```
yarn test
```
